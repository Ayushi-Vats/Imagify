# Imagify

Full-stack AI image generation app with authentication and credit system.

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB Atlas
- Razorpay (payments)

## Features
- User authentication
- Credit-based image generation
- Razorpay payment integration

## Run Locally

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm run dev  

## Environment Variables
Create a `.env` file in server:

MONGODB_URI=  
JWT_SECRET=  
RAZORPAY_KEY_ID=  
RAZORPAY_KEY_SECRET=  
CURRENCY=INR
